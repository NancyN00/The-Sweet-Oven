# The Sweet Oven

A bakery application with donuts, cakes, cupcakes and pancakes.

## Features

- User registration and login.
- Display items found.
- Update user profile.


## Technologies

The app uses these technologies;

- Kotlin - App is built with the language.
- Firebase- For user authentication.
